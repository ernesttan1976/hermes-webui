# Hermes Word Morphology Matrix – Reference Notes

This file contains compact reference notes for Hermes Agent when using the `hermes-word-morphology-matrix` skill.

## Common English Derivational Affixes (Short List)

- Prefix **un-**: negation, opposite of (`un-happy`, `un-kind`)
- Prefix **re-**: again (`re-teach`, `re-write`)
- Prefix **over-**: excessively, too much (`over-teach`, `over-cook`)

- Suffix **-ness**: forms abstract nouns meaning state or quality from adjectives (`happy → happi-ness`, `kind → kind-ness`)
- Suffix **-er**: agentive noun (person/thing that does the action) (`teach-er`, `run-ner`); also comparative on adjectives (`tall-er`) – treat these separately in explanations.
- Suffix **-est**: superlative of adjectives (`happi-est`, `tall-est`).
- Suffix **-ly**: forms adverbs from adjectives (`happi-ly`, `quick-ly`).
- Suffix **-able / -ible**: capable of being (`teach-able`, `read-able`).

## Spelling Patterns to Mention When Relevant

- Final **y → i** before `-ness`, `-ly`, `-er`, `-est`:
  - `happy → happi-ness`, `happi-ly`, `happi-er`, `happi-est`.
- Consonant doubling with `-ing`, `-ed` when needed (if used for a given base word).
